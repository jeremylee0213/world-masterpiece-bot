Nanobanana Prompt Studio

명화를 인기 캐릭터 감성으로 간단하게 리믹스

텍스트 대신 예시 이미지를 보며 명화를 선택하세요. 캐릭터와 배경만 고르면 자동 제안이 생성되고, 한 줄 프롬프트를 바로 만들 수 있습니다.

간편 설정

키 입력 → 모델 선택 → 명화/캐릭터/배경 선택 → 생성

원본 명화 선택
좌우로 넘겨서 확인하고 액자를 눌러 선택하세요. 모바일에서는 스와이프가 가능합니다.

생성 결과

한 줄 프롬프트를 복사해 이미지 모델에 바로 사용하세요.

생성 이미지

한 줄 프롬프트로 이미지를 생성할 수 있습니다.

이미지 다운로드

캐릭터 예시: 귀멸의 칼날, 도라에몽, 헌터헌터, 드래곤볼, 원피스, 나루토, 주술회전, 명탐정 코난, 원펀맨. 원하지 않으면 아래 입력창에서 캐릭터를 직접 입력해도 됩니다. 명화 미리보기 이미지는 Wikimedia 공개 자료 기반 URL을 사용하며, Nanobanana Pro 최고 해상도 출력 문구가 자동 포함됩니다.
